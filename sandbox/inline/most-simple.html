Most Simple Example Ever

This page is configured with the easiest installation method. It will default using the identity: noPii and trackers: page, but you can also still call custom tags as needed.
You can install this example, just by adding this onto your web page 

<script type="text/javascript" 
id="beaconScript"
async 
src="beacon.js"
data-api-root="http://localhost"
></script>
